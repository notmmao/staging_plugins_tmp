Let customers upload files before checkout in your WooCommerce webshop.

== Description ==

The WooCommerce Uploads Before Add-On is an add-on for our WooCommerce Uploads plugin and allows customers to upload one or more files for products before the checkout process.

== Installation ==

1. Make sure WooCommerce and WooCommerce Uploads are installed.
1. Download the .zip file.
1. Upload the entire ‘woocommerce-uploads-before’ directory into the ‘/wp-content/plugins/’ directory.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
1. Configure the plugin by going to WooCommerce -> Uploads (see ‘Getting started’ on this page for more information).
1. Enter the license key for this plugin in our WP Fortune plugin to get support and updates.
